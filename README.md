# Social Dashboard - Produccion

Este repositorio contiene la version compilada y lista para produccion de **Social Dashboard**.

## Estructura
* La raiz del repositorio contiene los archivos estaticos compilados del **Frontend (React)**.
* El directorio backend/ contiene el codigo del **Backend (PHP 8.4)**.

Para mas detalles sobre la instalacion y configuracion del entorno de base de datos, Apache/Nginx y permisos, consulte el archivo [DEPLOYMENT.md](./DEPLOYMENT.md).
